* 对自我认知上会出现问题（如认为自己是由 OpenAI 开发的等）
* 会“随便”生成图片（即用户的命令回复是一张图片等）
* 会对用户有冒犯性的语句
* 生成的图片未加限制（会出现 NSFW img）
